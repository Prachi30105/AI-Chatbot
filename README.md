# AI-Powered Chatbot with CI/CD

## Overview

An AI-powered chatbot developed using Python and Flask that provides intelligent conversational responses through REST APIs.

## Features

* Natural Language Processing
* REST API Integration
* Python Backend
* GitHub Version Control
* CI/CD using GitHub Actions
* Cloud Deployment Ready
* Scalable Architecture

## Tech Stack

* Python
* Flask
* OpenAI API
* Git & GitHub
* GitHub Actions
* Docker
* AWS / Azure

## Future Enhancements

* User Authentication
* Chat History Storage
* Kubernetes Deployment
* Monitoring and Logging
